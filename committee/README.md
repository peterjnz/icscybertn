# NZ ICS/OT Cyber Technical Network - Committee
The committee is made up of volunteers. The committee meets ~quarterly.

Committee responsibilities include
 - Attending and contributing to committee meetings
 - Providing steering and direction for the group
 - Supporting group forums (facilitating, identifying presenters, etc.)
 - Planning the annual summit (variety of tasks)
 - Providing other support and/or facilitating outcomes aligned to TN requirements
